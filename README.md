## Quick start

Quick start options:

- fork this repo
- git clone <repo link>

## Terminal Commands

1. Download and Install NodeJs LTS version from [NodeJs Official Page](https://nodejs.org/en/download/).
2. Navigate to the root ./ directory of the product and run `yarn install` or `npm install` to install the local dependencies.
3. Run `npm start` to run on your localhost.
4. Commit changes and raise your PR
